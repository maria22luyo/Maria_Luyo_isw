¿Qué son HTML, CSS y JavaScript?

HTML

HTML es el lenguaje de marcado que estructura el contenido de las páginas web. Define elementos como títulos, párrafos, imágenes y enlaces.

CSS

CSS es el lenguaje que da estilo a las páginas web. Permite cambiar colores, fuentes, tamaños y adaptar el diseño a distintos dispositivos.

JavaScript

JavaScript es el lenguaje de programación que da interactividad y dinamismo a una página. Permite crear efectos, validar formularios y mucho más.
