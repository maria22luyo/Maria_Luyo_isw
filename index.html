HTML, CSS y JavaScript

HTML

Se utiliza para la creación de una página web y lo que contiene. Este lenguaje estándar da estructura al contenido que un usuario busca en internet.

CSS

Se aplica a cualquier proyecto web que necesite diseño. Este lenguaje permite crear sitios adaptables, temas de color, tipografías y presentaciones limpias.

JavaScript

Lenguaje de programación que los desarrolladores web utilizan para crear interacciones dinámicas al desarrollar sitios web, aplicaciones, servidores y juegos.
